# AI Face Emotion & Persona HUD

A real-time computer vision project that detects facial features using MediaPipe and infers emotions and personas with a live cyber-style HUD overlay.

## Features
- Live webcam face tracking
- Emotion & persona inference
- Smooth emotion transitions
- Neon HUD interface
- Confidence meter
- FPS counter
- Emotion logging to CSV
- Fullscreen cinematic mode

## Tech Stack
- Python 3.10
- OpenCV
- MediaPipe
- NumPy

## How to Run
```bash
python emotion_hud.py
Press Q to quit, F to toggle fullscreen.
